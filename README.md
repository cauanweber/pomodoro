# Pomodoro Fullstack App

A fullstack Pomodoro application built with React, TypeScript, Node.js and PostgreSQL.

This project was created as a portfolio piece to demonstrate practical fullstack development, authentication, data persistence and production deployment.

### Frontend
- React
- Vite
- TypeScript
- Tailwind CSS (planned)

### Backend
- Node.js
- Express
- TypeScript
- PostgreSQL
- Prisma ORM
- JWT Authentication
